== Quick Start

=== Prerequisites

* Snowflake account with ACCOUNTADMIN privileges
* link:https://docs.snowflake.com/en/developer-guide/snowflake-cli/installation/installation[Snowflake CLI] installed
* link:https://docs.snowflake.com/en/user-guide/data-integration/openflow/setup-openflow-spcs[OpenFlow SPCS] deployed in your account

=== Installation

. *Clone the repository*
+
[source,bash]
----
git clone https://github.com/Snowflake-Labs/unified-data-stack-for-incident-management.git
cd unified-data-stack-for-incident-management
----

. *Configure environment*
+
[source,bash]
----
cp .env.template .env
# Edit .env with your configuration
----

. *Run automated setup*
+
[source,bash]
----
make install ENV_FILE=.env CONN=<your-connection-name>
----

. *Configure Slack connector*
+
Follow the link:md-docs/SETUP.adoc#slack-connector-configuration[detailed setup guide] to connect your Slack workspace.

*That's it!* You now have a fully functional AI-powered incident management platform.

For detailed installation instructions, see the link:md-docs/SETUP.adoc[*Setup Guide*].
